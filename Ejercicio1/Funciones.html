JavaScript Math

Metodo Math.abs()

Math.abs() devuelve el valor absoluto de un número:

document.getElementById("demo").innerHTML = Math.abs(-7.25);

Metodo Math.round()

El método Math.round() redondea un número al entero más cercano: 2,49 se redondeará hacia abajo (2) y 2,5 se redondeará hacia arriba (3).

document.getElementById("round").innerHTML = Math.round(2.60);

document.getElementById("round").innerHTML = Math.round(2.49);

Metodo Math.ceil()

El método Math.ceil() redondea un número redondeado hacia ARRIBA al entero más cercano.

document.getElementById("ceil").innerHTML = Math.ceil(0.50)

document.getElementById("ceil1").innerHTML = Math.ceil(-9.5)

Metodo Math.floor()

El método Math.floor() redondea un número HACIA ABAJO al entero más cercano.

document.getElementById("floor").innerHTML = Math.floor(0.50)

document.getElementById("floor1").innerHTML = Math.floor(-9.5)

Metodo Math.random()

El método Math.random() devuelve un número aleatorio entre 0 y 1:

document.getElementById("random").innerHTML = Math.random();

document.getElementById("random1").innerHTML = Math.random();

Metodo Math.exp()

El método Math.exp() devuelve ex, donde x es el argumento, y e es El número de Euler (también conocido como la constante de Napier), la base de los algoritmos naturales.

document.getElementById("exp").innerHTML = Math.exp(1);

document.getElementById("exp1").innerHTML = Math.exp(5);

Metodo Math.log()

El método Math.log() devuelve la base neutral de un número (base e)
